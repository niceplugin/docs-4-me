# [핵심개념.Blade컴포넌트] 개요
## 개요 {#overview}

Filament 패키지는 모든 인터페이스에 일관되고 유지 관리가 쉬운 기반을 제공하기 위해 핵심 Blade 컴포넌트 집합을 사용합니다. 이 컴포넌트들 중 일부는 여러분의 애플리케이션이나 Filament 플러그인에서도 사용할 수 있습니다.
